# Vue导出PDF

## html2pdf.js

```shell
sudo cnpm install --save html2pdf.js
```




## print-js

```shell
sudo cnpm install print-js   
```






## html2canvas & jspdf

```shell
sudo cnpm install --save html2canvas;
sudo cnpm install --save jspdf
```
