# vue 项目npm run dev  ip访问

> webpack `npm run dev` 不能通过ip访问
> 只能通过localhost访问

**解决方法如下：**

修改`vue-cli`: `config/index.js` 文件

把文件中 `host` 的值，改成 `ip` 即可
